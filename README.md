# TONFOLIO

> Non-custodial TON portfolio tracker & alerts — Telegram bot + Mini App.
>
> Некастодіальний трекер криптопортфеля на TON з алертами — Telegram-бот + Mini App.

**Non-custodial. Read-only. Ваші ключі ніколи не покидають гаманець.**

Проект у розробці — повна документація зʼявиться після завершення основних фаз.

## Quick start (dev)

```bash
docker compose up -d
pnpm install
pnpm db:push && pnpm db:seed
```
